# Portfolio Static Images

Drop image assets here. They are served at `/static/images/<filename>`
and referenced in templates with `{% static 'images/<filename>' %}`.

## Required

- **`profile.jpg`** — square headshot used on the home page intro section.
  Recommended: 600x600 or larger, JPG or PNG. If absent, the page falls
  back to a placeholder showing the initials "SS".
